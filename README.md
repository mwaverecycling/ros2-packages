# MWAVE ROS2 Packages

Assorted packages for ROS2 utilized by MWave Recycling

## Dependencies
Install the i2cpp libraries:
`git clone https://github.com/mwaverecycling/I2CPP.git`
`cd i2cpp/`
`cmake .`
`sudo make install`

## Install
`git clone https://github.com/mwaverecycling/ros2-packages.git'
`ament build`
